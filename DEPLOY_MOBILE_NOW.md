## ❓ Troubleshooting

### Website loads but blank?
- Wait 30 seconds for assets to load
- Clear browser cache (Ctrl+Shift+Delete)
- Refresh page

### Images not loading?
- Check internet connection
- Refresh page
- They load from Unsplash

### Mobile looks weird?
- Rotate phone to landscape then portrait
- Refresh browser
- Check Wi-Fi signal

### Mobile GPS not working?
- Allow location permission when prompted
- Turn on location services in phone settings
- Try manual address entry instead
