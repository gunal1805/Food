## 🚀 4 SIMPLE STEPS

### STEP 1️⃣: Go to GitHub
**Website:** https://github.com/

- Sign up for FREE account
- Or login if you have one

### STEP 2️⃣: Create Repository
After login on GitHub:

1. Click **"+"** icon (top right)
2. Click **"New repository"**
3. Name it: `ggs-boys-website`
4. Make it **PUBLIC**
5. Click **"Create repository"**

### STEP 3️⃣: Upload Your Files
On your new GitHub repository page:

1. Click **"upload files"** (or **"Add file"** → **"Upload files"**)
2. **Drag and drop** your entire folder:
   ```
   c:\Users\HP\OneDrive\Desktop\ggs-boys-website\
   ```
3. Or select files manually
4. Click **"Commit changes"**

### STEP 4️⃣: Deploy on Railway
**Website:** https://railway.app/

1. Click **"Create New Project"**
2. Click **"Deploy from GitHub"**
3. Click **"Connect GitHub and Select Repository"**
4. **Login to GitHub** when asked
5. Select `ggs-boys-website`
6. Click **"Create"** or **"Deploy"**
7. **WAIT 2-3 MINUTES** ⏳
